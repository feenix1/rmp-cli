# RMP CLI

Find everything you need on the [Rate My Professor](www.ratemyprofessors.com) website from the comfort of your own terminal.

## Quick Start

Navigate to the Releases page and install the latest version. Type `rmp help` for a list of commands.

## Features
**Currently in Alpha state, subject to change.**
- Persistent configuration for school details
- Filters searches to your specific school
- Get score and # of ratings for professors by name
- Supports opening professor's web page for more details
